Notebooks for Data Exploration

Technologies: Python 3.7.12, pandas 1.3.5, pyspark 2.4.5
Source dataset: CTD_chem_gene_ixns.csv

Notebooks list:
- check_actions.ipynb --> check the presence of selected actions;
- data_exploration_protein_molecule_statistics_HomoSapiens.ipynb --> statistics about Homo Sapiens;
- data_exploration_protein_molecule_statistics_MusMusculus.ipynb --> statistics about Mus Musculus;
- data_exploration_protein_molecule_diseases.ipynb --> molecule-disease association by instersecating molecules between sources (CTD_chem_gene_ixns.csv & CTD_curated_chemicals_diseases.csv)
- interactions.ipynb --> interactions analysis;
- species.ipynb --> statistics about species
